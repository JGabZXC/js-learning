Object Method

Methods are used to perform actions on the object or to manipulate its properties. They can be defined directly in the object literal or added later using dot or bracket notation.

Tip: Any function that is attached in object is called method

Accessing the calcAge property or method

                        
                            gab.calcAge(2003); // Dot Notation
                            gab["calcAge"](2003); // Bracket Notation

Making the calcAge calculate the gab.birthYear instead of typing it in arguments

                        
                            // Change the calcAge method in gab object
                            calcAge: function() {
                                return 2037 - this.birthYear; // The 'this' refers to that object(gab)
                            };
                        
                    
                        
                            console.log(gab.calcAge());

We can also add a property using this

Syntax

                            
                                calcAge: function(birthYear) {
                                    this.age = 2037 - birthYear; // Create new property
                                    return this.age;
                                }

Challenge

Create a method called getSummary and this method should return a string that summarize the data about gab

Tip: Use a Conditional(Ternary) operator

                        
                            // If hasDriversLicense is true
                            gab is a 21-year old student and he has a driver license // Output should be like this
                            // If hasDriversLicense is false
                            gab is a 21-year old student and he has no driver license // Output should be like this

Code for the Challenge

                            
                                const gab = {
                                    firstName: 'gab',
                                    lastName: 'lopez',
                                    birthYear: 2003,
                                    job: 'student',
                                    hasDriversLicense: false, // Change to true to check if hasDriversLicense
                                    calcAge: function() {
                                        this.age = 2024 - this.birthYear;
                                        return this.age;
                                    },
                                
                                    // Add getSummary method
                                    getSummary: function() {
                                        return `${this.firstName} is a ${this.calcAge()}-year old ${this.job} and he has ${this.hasDriversLicense ? 'a' : 'no'} driver license`;
                                    }
                                };
